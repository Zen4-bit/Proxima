INFO Architecture Overview

Proxima binds locally to 127.0.0.1 to keep your credentials and code context 100% private.

[Electron Main Process] (Core Router)
  ├── TCP IPC Port 19222 (MCP Bridge) <──> Coding Assistants (Cursor, VS Code, Windsurf)
  ├── HTTP Port 3210 (OpenAI REST API) <──> Custom scripts / CLI / SDKs
  └── WebSockets (ws://localhost:3210/ws) <──> Real-time streaming clients

🔒 Security & Key Storage Architecture

Session Mode Security

Operates via local Chromium instances. Session cookies and login tokens are stored locally on your machine.

Credentials and tokens never leave your system. Direct encrypted connections are made to official AI provider domains.

API Key Encryption

All Bring Your Own Key (BYOK) secrets are encrypted on-disk using hardware-bound encryption (DPAPI) via the safeStorage API.

Once loaded, requests bypass browser views and communicate directly with raw endpoints, guaranteeing zero telemetry and privacy.

💡 Why Proxima Exists ▸

In late 2025, I hit a massive roadblock with AI coding tools. Due to outdated training data, coding models would frequently hallucinate, guess wrong solutions, and end up ruining the source code and overall code quality. To fix this, I tried setting up local agents using Model Context Protocol (MCP) APIs, but the results were highly disappointing. The API responses were low-quality and frustratingly inaccurate, all while draining money for every single API call.

That’s when a wild idea struck: What if we could turn actual Web Models—like ChatGPT, Gemini, Claude, and Perplexity—into local MCP servers? By tapping directly into web interfaces via free logged-in accounts, I could give my local agents access to high-quality, up-to-date reasoning without paying for expensive and subpar API keys.

Proxima was born as a purely personal utility. I used it heavily for months to streamline my own work, with absolutely no plans of open-sourcing it. However, seeing so many developers online struggling with the exact same API costs and low-quality hallucinations, I decided to share it with the world. Proxima exists to empower developers with absolute privacy, speed, and high-quality local workflows—without forcing anyone to buy expensive software or pay for unreliable API keys.
